# Lokra_Assignment

## Overview

This project builds an end-to-end analytics workflow using SQL and Power BI to analyze customer behaviour, revenue trends, product performance, and conversion funnel efficiency.

The solution follows a layered data model:

RAW → STAGING → ANALYTICS → DASHBOARD

The final output is an interactive Power BI dashboard showing key business insights.

---

## Platform Choice

### Azure Synapse Analytics
Chosen because:

- Supports SQL-based transformations
- Works well with external tables (data lake architecture)
- Scalable analytics engine
- Easy integration with Power BI
- Supports structured layered modelling (staging → analytics)

### Power BI
Chosen because:

- Strong visualization capabilities
- Supports DAX calculations
- Interactive filtering
- Easy sharing and export options
